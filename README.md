# 💻 Diagnóstico Rápido - Suporte TI V1.0

Script simples em Python para diagnóstico básico de sistemas.

## 🔧 Funcionalidades:
- Informações do sistema
- Uso de CPU e memória
- Teste de conexão com internet

## 🚀 Como usar:
1. Instale dependência:
pip install psutil

2. Execute:
python diagnostico.py

## 🎯 Objetivo:
Auxiliar profissionais de suporte técnico em diagnósticos rápidos.
